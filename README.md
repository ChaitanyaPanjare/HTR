# ✍️ Handwritten Text Recognition (HTR)

## 📌 Overview
This project focuses on recognizing handwritten text using Deep Learning techniques.

## 🚀 Features
- Image preprocessing
- Model training using CNN + RNN / CRNN
- Prediction of handwritten text
- Evaluation using character & word accuracy

## 📊 Results
- Character Accuracy: 95.94%
- Word Accuracy: 92.64%
- Loss reduced from 11.83 → 0.16
